# Cupid's Canvas

A Valentine-themed interactive storybook with an animated town square splash screen. Built with HTML5 and CSS3.

## features

- **valentine town square splash screen** — animated scene with cupid, pedestrians, birds, balloons, heart confetti, and a couple on a bench
- **three-chapter storybook** — beginning, garden, and romance
- **pure css animations** — no JavaScript required
- **glassmorphism design** — modern translucent cards and effects
- **3d flip card** — interactive love letter with CSS envelope
- **responsive layout** — works on mobile and desktop
- **accessibility** — respects `prefers-reduced-motion`

## run locally

Open `index.html` in a web browser.

## tech stack

- HTML5
- CSS3 (keyframes, transforms, custom properties)
- Google Fonts (Great Vibes, Poppins)
